  3. Ganti file startup ke firmware baru
    Atur firmware baru sebagai startup. 
    <CIX-DC-PTG02-SW1> startup system-software flash:/CE6860EI-V200R019C10SPC800.cc all
Info: Operating, please wait for a moment........................................................done.
Info: Succeeded in setting the next startup software in slot 1.
  4. Hapus patch yang ada (jika perlu)
    Jika firmware baru tidak memerlukan patch, hapus patch yang ada. 
    <CIX-DC-PTG02-SW1> patch delete all
This will delete the patch. Are you sure? [Y/N]: y
Info: Operating, please wait for a moment...................done.
Info: Succeeded in deleting the patch.
