MPLS L2VC HUAWEI

Fungsi

Menghubungkan 2 node dengan tunneling, pertama pastikan fitur routing dan mpls pada switch huawei aktif

Perintah mpls l2vc digunakan untuk membuat Martini VLL.

Perintah undo mpls l2vc menghapus Martini VLL dari sebuah interface.

Secara default, tidak ada koneksi L2VPN Martini yang dibuat.

Format

mpls l2vc { ip-address | pw-template pw-template-name } * vc-id [ group-id group-id | tunnel-policy policy-name | [ control-word | no-control-word ] | [ raw | tagged ] | mtu mtu-value | [ secondary | bypass ] | ignore-standby-state ] *
undo mpls l2vc { ip-address | pw-template pw-template-name } * vc-id [ group-id group-id | tunnel-policy policy-name | [ control-word | no-control-word ] | [ raw | tagged ] | mtu mtu-value | [ secondary | bypass ] | ignore-standby-state ] *
undo mpls l2vc [ secondary | bypass ]

Parameter

Parameter Deskripsi Nilai
ip-address Menentukan ID LSR dari perangkat peer pada PW. Format desimal bertitik.
pw-template pw-template-name Menentukan nama template PW. String 1 hingga 19 karakter sensitif terhadap huruf besar, tidak mendukung spasi.
vc-id Menentukan ID L2VC. Integer dari 1 hingga 4294967295.
group-id Menentukan ID grup VC. Integer dari 1 hingga 4294967295.
tunnel-policy policy-name Menentukan nama kebijakan terowongan. String 1 hingga 39 karakter sensitif terhadap huruf besar, tidak mendukung spasi.
control-word Mengaktifkan fungsi kata kontrol. -
no-control-word Menonaktifkan fungsi kata kontrol. -
mtu mtu-value Menentukan nilai MTU. Integer dari 46 hingga 9600 (default 1500).
secondary Menentukan VC sekunder. -
bypass Menentukan VC bypass. -
ignore-standby-state Menentukan PW mengabaikan informasi status siaga yang dikirim oleh perangkat remote. -

Panduan Penggunaan

Dalam jaringan VLL Martini, Anda dapat menjalankan perintah ini untuk membuat koneksi VC.

Perhatian: Sebuah interface tidak dapat berfungsi sebagai L2VPN AC dan L3VPN AC secara bersamaan.

Contoh

# Buat koneksi Martini pada interface VLANIF.
 system-view
[HUAWEI] vlan batch 10
[HUAWEI] interface vlanif 10
[HUAWEI-Vlanif10] mpls l2vc 10.2.2.9 100

# Buat koneksi Martini pada GE0/0/1.
 system-view
[HUAWEI] interface gigabitethernet 0/0/1
[HUAWEI-GigabitEthernet0/0/1] undo portswitch
[HUAWEI-GigabitEthernet0/0/1] mpls l2vc 10.2.2.9 100
