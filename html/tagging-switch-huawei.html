Tagging VLAN pada Huawei Switch

1. Konfigurasi VLAN Access

Mode Access digunakan untuk port yang terhubung ke perangkat akhir (seperti komputer atau printer). Port ini hanya bisa mengakses satu VLAN.

Langkah-langkah:

  1. Masuk ke Mode Konfigurasi: 
    system-view
  2. Buat VLAN (jika belum ada): 
    vlan batch 10
  3. Konfigurasi Port sebagai Access: 
    interface GigabitEthernet 0/0/1
port link-type access
port default vlan 10

2. Konfigurasi VLAN Trunk

Mode Trunk digunakan untuk port yang menghubungkan perangkat jaringan (seperti switch ke switch) dan dapat mengangkut beberapa VLAN.

Langkah-langkah:

  1. Masuk ke Mode Konfigurasi: 
    system-view
  2. Buat VLAN (jika belum ada): 
    vlan batch 10 20
  3. Konfigurasi Port sebagai Trunk: 
    interface GigabitEthernet 0/0/2
port link-type trunk
port trunk allow-pass vlan 10 20

3. Konfigurasi VLAN Hybrid

Mode Hybrid adalah kombinasi antara mode Access dan Trunk. Port hybrid dapat mengakses VLAN tertentu sebagai access dan juga dapat mengangkut traffic dari VLAN lain.

Langkah-langkah:

  1. Masuk ke Mode Konfigurasi: 
    system-view
  2. Buat VLAN (jika belum ada): 
    vlan batch 10 20
  3. Konfigurasi Port sebagai Hybrid: 
    interface GigabitEthernet 0/0/3
port link-type hybrid
port hybrid allow-pass vlan 10
port hybrid pvid vlan 20

Catatan

  • Access: Hanya dapat mengakses satu VLAN (VLAN default).
  • Trunk: Dapat mengakses banyak VLAN dan sering digunakan untuk koneksi antar switch.
  • Hybrid: Dapat mengakses VLAN tertentu sebagai Access dan mengizinkan VLAN lain sebagai Trunk.
