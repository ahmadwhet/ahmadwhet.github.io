Konfigurasi VSI dan PEER pada Huawei Switch CLI

Langkah 1: Buat VSI

1. Masuk ke mode konfigurasi global:

system-view

2. Buat VSI dengan nama tertentu:

vsi <vsi-name> static

Contoh:

vsi VPLS_TEST static

Langkah 2: Konfigurasi Peer (LDP Signaling)

Peer digunakan untuk menghubungkan ke PE (Provider Edge) lain melalui LDP.

1. Masuk ke mode VSI lagi:

vsi VPLS_TEST static

2. Tambahkan PEER ke VSI:

peer <peer-ip> ldp

Contoh untuk PE dengan IP 192.168.1.1Sehingga akan seperti ini

vsi VPLS_TEST static
  peer 192.168.1.1 ldp

Langkah 3: Konfigurasi Dot1q Termination dan VSI

Langkah ini akan membantu Anda untuk melakukan konfigurasi Dot1q termination di interface dan menghubungkannya ke VSI.

Langkah-langkah

Buat Dot1q termination di interface:

        [Huawei] interface GigabitEthernet0/0/2
[Huawei-GigabitEthernet0/0/2] dot1q termination vid 200
[Huawei-GigabitEthernet0/0/2] vsi VPLS_TEST
[Huawei-GigabitEthernet0/0/2] quit

Penjelasan

  • dot1q termination vid 200Mengakhiri Dot1q pada VLAN ID 200.
  • vsi VPLS_TESTMenambahkan interface ke dalam VSI yang bernama VPLS_TEST.

Langkah 4: Verifikasi Konfigurasi

Untuk memastikan bahwa VSI dan peer telah dikonfigurasi dengan benar, Anda bisa menggunakan perintah berikut:

  • Cek status VSI: 
    display vsi all
  • Cek status LDP peer: 
    display ldp session
  • Cek status L2VPN: 
    display l2vpn vsi <vsi-name>
