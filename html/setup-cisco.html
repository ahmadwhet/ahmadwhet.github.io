SETUP SWITCH CISCO NEXUS

Untuk mengatur (setup) switch Cisco melalui CLI menggunakan koneksi console, Anda memerlukan perangkat seperti kabel console (biasanya kabel rollover atau kabel USB-to-RJ45) dan software terminal seperti PuTTY, Tera Term, atau HyperTerminal. Berikut langkah-langkahnya:

Persiapan

  1. Sambungkan Kabel Console:
    • Hubungkan ujung RJ45 kabel console ke port console di switch Cisco.
    • Hubungkan ujung lainnya ke port serial di komputer atau ke port USB (jika menggunakan adaptor USB-to-RJ45).
  2. Buka Software Terminal:
    • Buka aplikasi terminal (misalnya, PuTTY atau Tera Term).
    • Pilih koneksi Serial.
    • Atur Serial Line ke port COM yang sesuai (misalnya, COM3, COM4).
    • Atur Speed atau Baud rate ke 9600.
    • Klik Open untuk memulai sesi terminal.

Konfigurasi Dasar Switch

Setelah berhasil terhubung ke switch melalui console, berikut langkah-langkah untuk konfigurasi dasar:

  1. Masuk ke Mode Privileged:

Switch> enable

(Anda akan melihat prompt berubah menjadi Switch#)

  1. Masuk ke Mode Konfigurasi Global:

Switch# configure terminal

(Prompt akan berubah menjadi Switch(config)#)

  1. Atur Nama Host:

Switch(config)# hostname NamaSwitchAnda

(Misalnya, Switch(config)# hostname Switch1)

  1. Atur Password untuk Mode Privileged:

Switch(config)# enable password kata_sandi_anda

  1. Atur Password Konsol:

Switch(config)# line console 0

Switch(config-line)# password kata_sandi_konsol

Switch(config-line)# login

Switch(config-line)# exit

  1. Atur Password VTY (untuk akses Telnet/SSH):

Switch(config)# line vty 0 4

Switch(config-line)# password kata_sandi_vty

Switch(config-line)# login

Switch(config-line)# exit

  1. Atur IP Address pada VLAN 1 (untuk akses jaringan):

Switch(config)# interface vlan 1

Switch(config-if)# ip address 192.168.1.2 255.255.255.0

Switch(config-if)# no shutdown

Switch(config-if)# exit

(Gantilah 192.168.1.2 dengan alamat IP yang sesuai dengan jaringan Anda)

  1. Atur Default Gateway (jika diperlukan):

Switch(config)# ip default-gateway 192.168.1.1

(Gantilah 192.168.1.1 dengan alamat IP router atau gateway jaringan Anda)

  1. Simpan Konfigurasi:

Switch# write memory


atau dengan cara

 

Switch# copy running-config startup-config

  1. Keluar dari sesi:

Switch# exit

Tips Tambahan:

  • Untuk memeriksa konfigurasi IP:

Switch# show ip interface brief

  • Untuk melihat konfigurasi yang tersimpan:

Switch# show running-config




Untuk menambahkan user login di switch Cisco melalui CLI, Anda bisa membuat username dengan password atau secret untuk mengamankan akses ke perangkat. Pengaturan ini memungkinkan Anda menggunakan username dan password saat login ke switch, baik melalui console, Telnet, atau SSH. Berikut langkah-langkahnya:

Menambahkan User Login pada Cisco Switch

1.     Masuk ke Mode Privileged:

Switch> enable

2.     Masuk ke Mode Konfigurasi Global:

Switch# configure terminal

3.     Tambahkan Username dengan Password: Gunakan perintah berikut untuk membuat user baru:

Switch(config)# username nama_user password kata_sandi role network-admin

Contoh:

Switch(config)# username admin password cisco123 role network-admin

Di sini, admin adalah username dan cisco123 adalah password yang akan diminta saat login.

4.     Tambahkan Username dengan Secret (opsional, lebih aman): Anda bisa menggunakan secret alih-alih password untuk keamanan yang lebih baik karena secret akan dienkripsi:

Switch(config)# username nama_user secret kata_sandi

Contoh:

Switch(config)# username admin secret cisco123

Ini akan menyimpan cisco123 dalam bentuk terenkripsi.

5.     Pastikan VTY Line Menggunakan Login Local: Agar username dan password yang telah dibuat dapat digunakan saat login melalui Telnet atau SSH, atur VTY line untuk menggunakan login dengan local user:

Switch(config)# line vty 0 4
Switch(config-line)# login local
Switch(config-line)# exit

6.     Simpan Konfigurasi: Simpan perubahan agar tetap ada setelah switch di-restart:

Switch# write memory

atau

Switch# copy running-config startup-config

Menguji Login dengan User Baru

Setelah menambahkan user login, Anda bisa menguji login dengan username dan password baru baik melalui console, Telnet, atau SSH.

Misalnya, jika Anda terhubung ke switch melalui Telnet:

Copy code
telnet 192.168.1.2

Saat diminta, masukkan username dan password yang sudah Anda buat (admin dan cisco123 pada contoh di atas).

Dengan langkah-langkah ini, Anda sudah berhasil menambahkan user login di switch Cisco menggunakan CLI.


Tips Tambahan:

  • Untuk enable feature telnet:

    Switch# configure terminal
    Switch(config)# feature telnet

  • Untuk enable feature interface vlan:

Switch# configure terminal
Switch(config)# feature interface-vlan

  • Untuk enable feature lacp / bonding:

Switch# configure terminal
Switch(config)# feature lacp

  • Untuk enable feature lldp:

Switch# configure terminal
Switch(config)# feature lldp
