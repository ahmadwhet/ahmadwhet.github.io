Reset Switch Force10 S4810 + S60

  1. Koneksikan USB serial dengan baud rate 9600.
  2. Reboot switch.
  3. Tekan tombol apa saja pada keyboard untuk masuk ke menu boot.
  4. Ketik syntax berikut:
BOOT_USER #setenv stconfigignore true
BOOT_USER #saveenv
BOOT_USER #reset

#done

Reset Switch Force10 S50N

  1. Koneksikan USB serial dengan baud rate 9600.
  2. Reboot switch.
  3. Tekan angka 1 atau 2 pada keyboard untuk masuk ke menu boot.
  4. Ketik syntax berikut:
BOOT_USER #restore factory-defaults
BOOT_USER #reload

#done

Reset Force10 S50

Bisa dilakukan dengan langkah berikut:

  1. Connect ke port console.
  2. Power cycle switch.
  3. Tekan Ctrl+C saat proses booting. Jika muncul prompt #, maka proses berhasil. Jika muncul =>, ulangi lagi.
  4. Gunakan perintah berikut untuk overwrite konfigurasi:
sysctl -w ddb.command="call flashStrataJ3CEraseSector(0x01a00000)"
sysctl -w ddb.command="call flashStrataJ3CEraseSector(0x01a40000)"
sysctl -w ddb.command="call flashStrataJ3CEraseSector(0x01a80000)"
sysctl -w ddb.command="call flashStrataJ3CEraseSector(0x01ac0000)"
sysctl -w ddb.command="call flashStrataJ3CEraseSector(0x01b00000)"
sysctl -w ddb.command="call flashStrataJ3CEraseSector(0x01b40000)"
sysctl -w ddb.command="call flashStrataJ3CEraseSector(0x01b80000)"
sysctl -w ddb.command="call flashStrataJ3CEraseSector(0x01bc0000)"
sysctl -w ddb.command="call flashStrataJ3CEraseSector(0x01c00000)"
sysctl -w ddb.command="call flashStrataJ3CEraseSector(0x01c40000)"
sysctl -w ddb.command="call flashStrataJ3CEraseSector(0x01c80000)"
sysctl -w ddb.command="call flashStrataJ3CEraseSector(0x01cc0000)"
  • Gunakan perintah berikut untuk restart switch:
    • sysctl -w ddb.command="call ssCpldBoardReset"

    Setelah selesai, switch akan kembali ke default tanpa konfigurasi console atau enable password.

    Cara Lain Reset Switch Force10 S4810 + S60

    1. Koneksikan console dengan baud rate 9600.
    2. Hidupkan switch dan login.
    3. Ketik syntax berikut:
    #restore factory-defaults stack-unit 0 clear-all
  • Ketik "yes" dan tunggu switch untuk restart otomatis.
  • Tekan tombol "A" saat masuk ke mode BMP/ACLS untuk melanjutkan booting.

    1. 1. konekkan console di bautrate 9600
    2. 2. hidupkan switch, login enable
    3. 3. ketik syntax berikut :
    #clear config
  • 4. Enter. Ketik "y / yes" dan switch akan tereset
  • 5. Untuk memastikan, restart switch nya
