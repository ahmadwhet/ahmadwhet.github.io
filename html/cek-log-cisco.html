Cara Cek Log dengan Detail pada Switch Cisco Nexus

1. Masuk ke Mode EXEC

Pastikan kamu sudah terhubung ke switch dan berada di mode EXEC. Jika belum, masuk ke mode EXEC dengan menggunakan perintah berikut:

enable

2. Mengaktifkan Logging

Jika logging belum diaktifkan, aktifkan di mode konfigurasi:

    configure terminal
    logging enable

3. Menampilkan Log

Gunakan perintah berikut untuk menampilkan log secara detail:

  • Menampilkan Log Terakhir: 
    show logging
  • Menampilkan Log dengan Tanggal dan Waktu: 
    show logging | include timestamp

4. Menampilkan Log dengan Filter

Gunakan filter untuk menampilkan log yang lebih spesifik:

show logging | include <severity-level>

Ganti <severity-level> dengan level yang diinginkan (misalnya: informational, warning, error).

5. Menampilkan Log Secara Real-Time

Untuk melihat log secara real-time, gunakan perintah berikut:

terminal monitor

Tekan Ctrl + C untuk menghentikannya.

6. Menyimpan Log ke File (Opsional)

Jika ingin menyimpan log ke file, gunakan perintah berikut:

show logging | redirect flash:log.txt

Ini akan menyimpan log ke dalam file log.txt di flash memory switch.

7. Memeriksa Log Spesifik

Untuk memeriksa log berdasarkan waktu atau peristiwa tertentu:

show logging | begin <keyword>

Ganti <keyword> dengan kata kunci yang relevan.

Contoh Penggunaan

Berikut adalah contoh perintah untuk melihat log:

show logging
show logging | include error
show logging | include 2024

Dengan menggunakan perintah-perintah di atas, kamu dapat memeriksa dan menganalisis log pada switch Cisco Nexus dengan detail. Jika ada pertanyaan lebih lanjut, jangan ragu untuk bertanya!
