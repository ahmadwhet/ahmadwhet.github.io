Konfigurasi VLAN pada Switch Cisco Nexus

1. Konfigurasi Access Port

Access port digunakan untuk menghubungkan perangkat yang hanya memerlukan satu VLAN (misalnya, PC). Port ini tidak akan mengirimkan tag VLAN.

Langkah-langkah:

    configure terminal
    interface Ethernet1/1
    switchport mode access
    switchport access vlan 10  # Ganti 10 dengan ID VLAN yang sesuai
    end
    write memory

2. Konfigurasi Trunk Port

Trunk port digunakan untuk menghubungkan switch dan dapat membawa beberapa VLAN. Port ini akan mengirimkan tag VLAN.

Langkah-langkah:

    configure terminal
    interface Ethernet1/2
    switchport mode trunk
    switchport trunk allowed vlan 10,20,30  # Ganti dengan ID VLAN yang sesuai
    end
    write memory

3. Verifikasi Konfigurasi

Setelah melakukan konfigurasi, kamu bisa memverifikasi dengan perintah berikut:

  • Untuk access port: 
    show running-config interface Ethernet1/1
  • Untuk trunk port: 
    show running-config interface Ethernet1/2
  • Untuk melihat status trunk: 
    show interface trunk

Dengan langkah-langkah di atas, kamu bisa mengkonfigurasi VLAN tagging pada switch Cisco Nexus sesuai kebutuhan. Pastikan untuk mengganti Ethernet1/1 dan Ethernet1/2 dengan interface yang sesuai pada switch kamu.
