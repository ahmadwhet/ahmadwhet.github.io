Konfigurasi QinQ pada Switch Cisco Nexus

Pengantar

QinQ (tunneling 802.1Q) memungkinkan banyak VLAN untuk dienkapsulasi di dalam VLAN lain, memungkinkan transportasi lalu lintas VLAN di seluruh jaringan tanpa kehilangan informasi VLAN asli.

Langkah-langkah untuk Mengonfigurasi QinQ

1. Masuk ke Mode Konfigurasi

enable
configure terminal

2. Membuat VLAN

Buat VLAN luar (S-VLAN) dan VLAN dalam (C-VLAN). Misalnya, gunakan VLAN 100 sebagai S-VLAN dan VLAN 10 sebagai C-VLAN:

vlan 100
  name S-VLAN

vlan 10
  name C-VLAN

3. Mengonfigurasi Interface Trunk

Pilih interface yang akan digunakan untuk QinQ dan atur ke mode trunk. Misalnya, menggunakan Ethernet 1/1:

interface ethernet 1/1
  switchport mode trunk
  switchport trunk allowed vlan 10,100
  switchport trunk native vlan 100
  switchport mode dot1q-tunnel

4. Mengaktifkan QinQ pada Interface

interface ethernet 1/1
  switchport qinq enable

5. Menyimpan Konfigurasi

end
write memory

6. Memverifikasi Konfigurasi

Gunakan perintah berikut untuk memverifikasi bahwa QinQ telah dikonfigurasi dengan benar:

show vlan brief
show interface trunk
show running-config interface ethernet 1/1
