Tagging VLAN pada Switch Cisco Nexus

Langkah-langkah Konfigurasi

  1. Masuk ke Mode Konfigurasi Global: 
    configure terminal
  2. Pilih Interface yang Akan Dikonfigurasi: 
    interface ethernet1/1
  3. Aktifkan Mode Trunk: 
    switchport mode trunk
  4. Tentukan VLAN yang Diizinkan: 
    switchport trunk allowed vlan 10,20,30
  5. Verifikasi Konfigurasi: 
    show interface ethernet1/1 switchport

Contoh Konfigurasi Lengkap

configure terminal

interface ethernet1/1
  switchport mode trunk
  switchport trunk allowed vlan 10,20,30  ! Ganti dengan VLAN yang sesuai

exit

Catatan Tambahan

Pastikan bahwa VLAN yang ingin Anda tambahkan telah dibuat sebelumnya. Anda dapat membuat VLAN menggunakan perintah berikut:

vlan 10
vlan 20
vlan 30

Setelah konfigurasi, pastikan untuk menguji koneksi untuk memastikan VLAN ditagging dengan benar di jaringan Anda.
