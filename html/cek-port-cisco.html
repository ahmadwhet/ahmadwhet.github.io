Cara Cek Port, SFP, dan Redaman pada Switch Cisco Nexus

1. Memeriksa Status Port yang Running

Gunakan perintah berikut untuk memeriksa status semua port:

show interface status

Output yang Diharapkan:

Port      Name               Status       Vlan       Duplex  Speed  Type
--------  ----------------  ------------  ---------- ------- ------- ------
Eth1/1                     connected      10         a-full  a-1000 1000BaseSX
Eth1/2                     notconnect     10         auto    auto   1000BaseSX
Eth1/3                     connected      20         a-full  a-1000 1000BaseLX

2. Memeriksa Detail Interface

Untuk mendapatkan detail lebih lanjut tentang interface tertentu:

show interface Ethernet1/1

Output yang Diharapkan:

Ethernet1/1 is up, line protocol is up 
  Hardware is Ethernet, address is 5254.00ff.1234 (bia 5254.00ff.1234)
  MTU 1500 bytes, BW 1000000 Kbit/sec, DLY 10 usec,
     reliability 255/255, txload 1/255, rxload 1/255
  Encapsulation ARPA, loopback not set
  Keepalive not configured
  Full Duplex, 1000 Mb/s
  Input flow-control is off, output flow-control is off

3. Memeriksa SFP (Small Form-factor Pluggable)

Untuk memeriksa informasi tentang SFP yang terpasang:

show interface transceiver details

Output yang Diharapkan:

Ethernet1/1
  Name                 : SFP-1000BASE-SX
  Type                 : 1000BASE-SX
  Wavelength           : 850 nm
  Length (m)          : 550
  Vendor ID            : CISCO
  Serial Number        : ABC12345
  Temperature          : 29 C
  Voltage              : 3.3 V

4. Memeriksa Redaman (Attenuation)

Untuk memeriksa redaman, gunakan perintah berikut:

show interfaces Ethernet1/1 status

Output yang Diharapkan:

Ethernet1/1 is up
  Input power: -10 dBm
  Output power: -5 dBm

5. Menampilkan Statistik SFP

Gunakan perintah ini untuk melihat statistik SFP:

show interface Ethernet1/1 status | include SFP

Output yang Diharapkan:

SFP is present
  Type: 1000BASE-SX
  Status: Active

Ringkasan

Dengan menggunakan perintah-perintah di atas, kamu dapat mendapatkan informasi yang komprehensif mengenai status port, detail SFP, dan redaman pada switch Cisco Nexus. Setiap output akan memberikan informasi penting untuk membantu mendiagnosis masalah atau memastikan kinerja yang optimal.
