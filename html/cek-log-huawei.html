Cara Cek Log pada Huawei Switch

  1. Masuk ke CLI Switch Huawei

    Hubungkan ke switch Huawei melalui telnet, SSH atau menggunakan kabel console.

  2. Masukkan Nama Pengguna dan Kata Sandi

    Masukkan username dan password yang telah diatur untuk akses switch.

  3. Masuk ke Mode User atau Administrator 
    system-view

    Gunakan perintah di atas untuk masuk ke mode system-view.

  4. Menampilkan Log Real-Time 
    terminal monitor

    Perintah ini akan menampilkan log langsung di layar Anda saat ada event baru yang terjadi.

  5. Melihat Log yang Tersimpan 
    display logbuffer

    Gunakan perintah ini untuk melihat buffer log yang menyimpan log terakhir dari switch.

  6. Melihat Log Alarm 
    display trapbuffer

    Perintah ini akan menampilkan pesan-pesan alarm yang telah terjadi di switch.

  7. Filter Log Berdasarkan Kriteria Tertentu 
    display logbuffer | include <keyword>

    Ganti <keyword> dengan kata yang ingin Anda cari, seperti "error" atau "warning".

  8. Menyimpan Log ke File 
    save logfile logbuffer.log

    Ini akan menyimpan buffer log ke dalam file logbuffer.log.

  9. Menampilkan Log dari File 
    display logfile logbuffer.log
  10. Menghapus Log (Jika Diperlukan) 
    reset logbuffer

    Perintah ini akan menghapus semua log yang ada di dalam log buffer.
