Tutorial Konfigurasi VXLAN dan Bridge Domain pada Switch Huawei

Persyaratan

  • Switch Huawei yang mendukung VXLAN.
  • Akses ke perangkat melalui konsol, telnet atau SSH.
  • Konfigurasi jaringan dasar (VLAN, IP address, dan routing).

1. Konfigurasi Nve

Langkah-langkah Konfigurasi VXLAN VNI

  1. Masuk ke Mode Konfigurasi: 
    system-view
  2. Buat Interface Nve: 
    Interface Nve1
  3. Mengonfigurasi Pseudo-Wire: 
    source 10.10.102.1
 vni 9510 head-end peer-list 10.10.102.2 // IP destination 
 vni 9510 head-end peer-list 10.10.102.10 // IP destination
 vni 9510 head-end peer-list 10.10.102.11 // IP destination

    2. Menambahkan Bridge Domain ke VXLAN

    Langkah-langkah Konfigurasi Bridge Domain

    1. Buat Bridge Domain: 
      bridge-domain 951
    2. Menambahkan VNI ke Bridge Domain: 
      bridge-domain 951
    vxlan vni 9510

      3. 3. Menambahkan Sub-Interface L2 Untagged ke Bridge Domain

      Langkah-langkah Konfigurasi Sub-Interface L2 Untagged

    3. Membuat Sub-Interface L2 Untagged: 
      interface GigabitEthernet 0/0/1.20 mode l2
 encapsulation dot1q vid 20
      vid 20 merupakan vlan yang akan didelivery
    4. Mengaitkan Sub-Interface dengan Bridge Domain: 
      bridge-domain 951
    5. Simpan Konfigurasi: 
      save

    Verifikasi Konfigurasi Sub-Interface L2 Untagged

    display bridge-domain
    Tampilan seperti ini

    Kesimpulan

    Dengan mengikuti langkah-langkah di atas, Anda telah berhasil mengonfigurasi VXLAN VNI, bridge domain, dan sub-interface L2 untagged pada switch Huawei. Pastikan untuk menyesuaikan ID bridge domain, VLAN, dan parameter lain sesuai dengan topologi jaringan Anda. Jika Anda memiliki pertanyaan lebih lanjut atau memerlukan bantuan tambahan, silakan tanyakan!
